# Project Documentation

This branch contains the Doxygen pages for the project. It can be accessed at https://tewi.aest.dev.

The Doxygen documentation is automatically generated by the GitHub Actions workflow defined in the `.github/workflows/doxygen.yml` file.

> [!CAUTION]
> This is not setup yet because of a bug in the Doxygen generation process. A broken interaction with C++ modules makes member and type definitions show twice in the documents. 

# Contributing

Do not edit the files in this branch directly, as they are generated from the source code in the `main` branch. Instead, open a pull request in the `main` branch with your changes in the source files and then the Doxygen generation process is automatically triggered when it is merged to the `main` branch. The generated documentation will be updated accordingly in this branch.
